***Когда делаешь новую строку, делай два пробела в конце абзаца или пустую строку между абзацами. Это нужно для GitHub Wiki. Если не понял, то посмотри в файлах как  сделаны новые строки (обрати внимание на всё пространство вокруг новой строки, там может быть не всё видно на первый взгляд)***

# Ссылка
Ссылка на заметку - [Шпора](/Шпаргалка.md) - \[Шпора](Шпаргалка.md)
Ссылка на заглавие - [Шпора](/Шпаргалка.md#Ссылка) - \[Шпора](Шпаргалка.md#Ссылка)
Ссылка вставка - !\[Шпора](Шпаргалка.md#Меньшее%20заглавие) (%20 - пробел в названии) *(не отображаться в GitHub)*![Шпора](/Шпаргалка.md#Меньшее%20заглавие)
Ссылку можно создать, перетянув нужный файл в поле редактора.

***Важно***
В настройках -> "Файлы и ссылки"
Формат новой ссылки -> Абсолютный путь относительно хранилища
Использовать \[\[Wiki-ссылки\]\] -> Выкл
Это нужно для возможности просматривать файлы на странице GitHub через Wiki
Когда делаешь ссылку, в начале части в () ставь /
